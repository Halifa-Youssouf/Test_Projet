# Dashboard d'Analyse des Ventes Globales (Financials)

# Contexte du projet
Ce projet présente un tableau de bord interactif réalisé avec Power BI, analysant les performances financières d'une entreprise sur plusieurs segments de marché et zones géographiques. L'objectif est d'identifier les leviers de croissance et de comprendre la répartition de la valeur par produit et par pays.

# Objectifs du projet
1.Visualiser la répartition des ventes par segment et par pays 

2.Identifier les produits les plus vendus par segment et par pays

3.Suivre l’évolution des ventes dans le temps

4.Fournir un outil interactif et intuitif pour la prise de décision

# Stack Technique
Outil : Microsoft Power BI Desktop.

Source de données : Dataset financier (Excel/CSV).

Langage : DAX (Data Analysis Expressions) pour les mesures calculées.

Compétences démontrées : Nettoyage de données (Power Query), Modélisation de données, Data Viz et Storytelling.
